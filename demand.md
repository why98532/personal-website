# 极简技术个人主页 / 作品集官网 – 详细开发说明书

## 1. 项目概述

**项目名称**：极简技术个人主页 / 个人作品集官网  
**视觉参考**：完全复刻 [https://benx.ai](https://benx.ai) 的视觉风格与页面结构（极简、深色模式、科技感、干净留白）  
**架构要求**：纯静态，无后端、无数据库、无接口、无登录、无后台管理  
**使用场景**：个人名片、职业背书、作品集展示、AI 技术干货分享、对外商务对接  
**核心调性**：极简、深色模式、科技感、干净留白、专业靠谱、轻量化、打开速度快

---

## 2. 硬性技术约束（必须严格遵守）

- ✅ 只使用前端静态页面（HTML/CSS/JS），不写任何后端代码
- ✅ 不使用任何数据库（MySQL、SQLite、云数据库等），不做任何数据存储
- ✅ 所有文案、经历、项目、文章全部写死在静态文件或 Markdown 文件中（无 CMS）
- ✅ 全站支持手机、电脑自适应（响应式布局）
- ✅ 默认深色主题，并提供一键切换浅色主题的功能
- ✅ 页面打开速度极快：无复杂动画、无冗余插件、无大图懒加载（图片尽量使用 SVG 或压缩图）
- ✅ 代码结构整洁，注释清晰，方便二次修改文案、头像、联系方式
- ✅ 可一键部署到 Vercel / Netlify / GitHub Pages / 阿里云 / 腾讯云等静态托管平台

## 3. 推荐技术栈（AI 严格按此选型）

| 类别         | 技术选型                                      |
|--------------|-----------------------------------------------|

**文章详情页**：

**文章管理方式**：
- 所有文章存放在 `posts/` 目录，每个文章一个 `.md` 文件
- 文件头部使用 Frontmatter 定义 `title`, `date`, `summary`, `tags`（可选）
- AI 需提供 2~3 篇示例文章（内容为占位文案，介绍 AI 技术或工具使用）

**搜索实现细节**：
- 在列表页客户端通过 `fetch` 获取所有文章的 Frontmatter + 摘要
- 使用 JavaScript 进行实时过滤（防抖）
- 不依赖任何外部搜索引擎

### 6.2 代码风格与注释要求

- 所有配置文件、组件、布局文件必须包含必要的中文或英文注释，说明用途
- CSS 优先使用 Tailwind 工具类，自定义样式放在 `custom.css` 中并注明用途
- 响应式断点使用 Tailwind 默认值：`sm` `md` `lg` `xl`
- 禁止使用内联样式，禁止引入外部非必要库（如 jQuery、Bootstrap）

### 6.3 主题切换实现细节

- 在 `html` 或 `body` 上添加 `.dark` class 控制深色主题
- 默认 `.dark` 生效（深色模式）
- 切换时更新 `localStorage.setItem('theme', 'dark'/'light')` 并移除/添加 class
- 页面加载时读取 `localStorage` 恢复用户选择
